# ArcheNavigator

ArcheRage 좌표를 지도에서 빠르게 확인하는 모바일 친화형 웹 도구입니다.

A mobile-friendly web tool for locating ArcheRage coordinates on an interactive map.

## 바로 사용하기 / Open the app

**[ArcheNavigator 실행 / Launch ArcheNavigator](https://rumor88.github.io/ArcheNavigator/)**

## 기능 / Features

### 한국어

- 동서(E/W) 및 남북(N/S) 좌표를 도·분·초 단위로 입력해 지도에 마커를 표시합니다.
- 지도를 직접 누르면 해당 위치로 마커가 이동하고 좌표가 팝업으로 표시됩니다.
- 마커 표시 시 입력한 위치로 지도를 자동 이동하고 보기 좋은 확대 수준으로 조정합니다.
- 좌표가 포함된 URL을 열면 해당 위치를 자동으로 불러옵니다.
- 스마트폰과 데스크톱 화면에 맞춘 반응형 UI를 제공합니다.

### English

- Enter east/west (E/W) and north/south (N/S) coordinates in degrees, minutes, and seconds to place a marker.
- Tap or click anywhere on the map to move the marker and view that location's coordinates in a popup.
- Placing a marker automatically centers the map and selects a useful zoom level.
- Open a URL containing coordinate parameters to load that location automatically.
- Use the responsive interface on both mobile and desktop screens.

## 사용 방법 / How to use

1. 동서·남북 방향과 도·분·초를 입력합니다. / Enter the directions, degrees, minutes, and seconds.
2. `마커 표시`를 누릅니다. / Select `마커 표시` to place the marker.
3. 지도 마커의 팝업에서 좌표를 확인합니다. / Read the coordinates in the marker popup.

The site is published with GitHub Pages from `index.html` on the `main` branch.
